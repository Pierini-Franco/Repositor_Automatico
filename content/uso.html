Modo de Uso

Para utilizar este proyecto, se emplea una página web diseñada para realizar pedidos, renovar o editar productos. A continuación, se describen los pasos para operar el sistema:

  • El usuario ingresa a la página web para realizar un pedido indicando el producto deseado y la cantidad.
  • El robot se activa automáticamente y se dirige a la ubicación del producto solicitado.
  • El robot transporta el producto a la base de recepción para que el usuario recoja la cantidad especificada.
  • Después de retirar los productos, el usuario debe presionar el botón rojo en el robot para indicarle que puede regresar el pedido.
  • Posteriormente, el robot vuelve a su base de espera.
  • Finalmente, tras presionar el botón de finalización, la página web actualiza automáticamente el valor del stock en el sistema.
